# MBEL：よくある質問 (FAQ)

### Q：なぜ単に JSON を使わないのですか？
**A：** JSON はデータ形式であり、言語ではありません。MBEL はプログラム可能なローカライズ言語であり、コメント、ロジックブロック、および AI メタデータをネイティブにサポートしています。

### Q：MBEL は私の言語の複数形ルールをサポートしていますか？
**A：** はい。MBEL には、日本語、英語、ドイツ語、フランス語、ポーランド語、ロシア語、スペイン語、中国語を含む 15 以上の主要な世界の言語の CLDR ルールが組み込まれています。

### Q：Go SDK はどのくらい速いですか？
**A：** 非常に高速です。MBEL はファイルを最適化された AST にコンパイルします。解決は単純な O(1) マップルックアップです。
