# MBEL: FAQ (Preguntas Frecuentes)

### P: ¿Por qué no usar simplemente JSON?
**R:** JSON es un formato de datos, no un lenguaje. MBEL jest un lenguaje de localización programable que soporta comentarios, bloques lógicos y metadatos de IA de forma nativa.

### P: ¿Soporta MBEL reglas de plural para mi idioma?
**R:** Sí. MBEL incluye reglas CLDR para más de 15 idiomas principales, incluyendo español, inglés, alemán, francés, polaco, ruso, chino y japonés.

### P: ¿Qué tan rápido es el SDK de Go?
**R:** Extremadamente rápido. MBEL compila los archivos en un AST optimizado. La resolución es una búsqueda simple en un mapa O(1).
