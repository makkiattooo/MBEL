# MBEL: FAQ (Часто задаваемые вопросы)

### В: Почему бы просто не использовать JSON?
**О:** JSON — это формат данных, а не язык. MBEL — это программируемый язык локализации, который нативно поддерживает комментарии, логические блоки и ИИ-метаданные.

### В: Поддерживает ли MBEL правила множественного числа для моего языка?
**О:** Да. MBEL включает правила CLDR для более чем 15 основных мировых языков, включая русский, английский, немецкий, французский, польский, испанский, китайский и японский.

### В: Насколько быстро работает Go SDK?
**О:** Чрезвычайно быстро. MBEL компилирует файлы в оптимизированный AST. Разрешение ключа — это простой поиск в хеш-таблице (map) за O(1).
